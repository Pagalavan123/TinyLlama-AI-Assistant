# TinyLlama QLoRA Fine-Tuning 🚀

This repository demonstrates how to fine-tune the TinyLlama 1.1B Chat model using QLoRA in Google Colab with low GPU memory usage.

---

## 📌 Topics Covered

- TinyLlama Fine-Tuning
- QLoRA (4-bit Quantization)
- Hugging Face Transformers
- PEFT / LoRA
- TRL SFTTrainer
- Instruction Dataset Formatting
- Memory Efficient LLM Training

---

## 🛠️ Tech Stack

- Python
- PyTorch
- Hugging Face
- Transformers
- PEFT
- BitsAndBytes
- TRL
- Google Colab

---

## 📂 Included

- Fine-tuning notebook
- QLoRA setup
- Dataset formatting
- Training configuration
- Model saving workflow

---

## 🎯 Goal

The main objective of this project is to explore efficient LLM fine-tuning techniques using low-resource hardware environments such as Google Colab.
